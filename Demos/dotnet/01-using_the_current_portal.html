Module 1: Overview of the Microsoft Azure Platform

Demo: Using the Current Azure Portal

  1. On the Start screen, click the Internet Explorer tile.

  2. Go to https://portal.azure.com.

  3. In the email address box, type the email address of your Microsoft account.

  4. Click Continue.

  5. In the password box, type your password for your Microsoft account.

  6. Click Sign In.

  7. If you are prompted with a dialog box that introduces you to the new portal, click Get Started.

  8. After the dashboard is fully loaded, right-click the Dashboard (home screen), and then click Edit Dashboard.

    You can use the settings of each tile to resize the tile and customize the Dashboard.

  9. Right-click the Dashboard (home screen), and then click Done Customizing.

  10. In the top-left corner of the screen, click New.

  11. In the New panel that displays, click See all.

  12. In the Everything blade that displays, click Compute.

  13. In the Compute blade that displays, click the Ubuntu Server option.

  14. In the Ubuntu Server blade that displays, click the latest version of Ubuntu Server LTS.

    The version number for Ubuntu Server is not important in this demonstration. Also, you might have to scroll down to find the Ubuntu Server option

  15. In the Ubuntu Server blade that displays, click the Create button.

  16. In the Basics blade that displays, locate the Name dialog box and type any value.

  17. Scroll to the left, and then click the close (X) button on the Create virtual machine blade to close the journey.

  18. In the Message from webpage Internet Explorer dialog box, click OK.
